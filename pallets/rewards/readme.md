## Rewards Pallet

Rewards Pallet is a module handling new block rewards, which is designed for Substrate based hybrid PoW + PoA (GRANDPA) chains. 
- Rewards split: 70% (miners)/ 30% (validators)
- Vesting schedule: after a block is produced, rewards are getting available to unlock by 10% every 10 days (in blocks) 
- Mining pool rewards handling
